### 功能列表

1.变量提示
2.函数提示
3.枚举提示
4.支持API文件查看
5.变量 函数  枚举的悬停提示
6.跳转到官方api

### 使用规则
    由于常量 变量类型 变量 函数 中,模糊查询过程中容易混淆,因此,给每一类增加前缀
    变量类型前缀:ty
    常量前缀:cn
    函数前缀:fn
    变量前缀:vr
